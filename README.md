To display your name on a new line, add 2 spaces to the end of the previous line.

_Example:_  
Contributors:\[ENTER\](*Your Name*) will display:  
<b>Contributors:(*Your Name*)</b>  

Contributors:\[SPACE\]\[SPACE\]\[ENTER\](Your Name) will display:  
<b>Contributors:  
(Your Name)</b>
